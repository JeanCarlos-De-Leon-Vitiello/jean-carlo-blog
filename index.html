Goals And Plans

What I want to achieve during the upcoming Web Design and Development minor.

Learning Goal 1

JavaScript

I want to improve my JavaScript skills so I can better understand how functions, events, and logic work in interactive websites. At the moment, I sometimes write code without fully understanding why it works. That is why I want to gain a deeper understanding of JavaScript structure and functionality, so I can build features and solve problems with more confidence and independence.

Learning Goal 2

CSS, Design, and Animation

I want to further develop my skills in CSS, design, and web animations so I can create websites that are not only functional, but also visually appealing and professional. I want to learn how to use animations in a subtle and effective way to improve the user experience. In addition, I want to improve my understanding of layouts, color usage, and modern design principles.
